Class GradientLinear

Class defining a 'linear' gradient 'window' that currently goes only from left to right. Specifically, a horizontal section of the canvas is visible with blending transitions on the left and right side of the window.

Note, when the gradient position extends beyond the right or left side it loops around to the other side to create a seamless animation.
